# Calculadora Avançada em Python

Uma *calculadora avançada* desenvolvida em Python usando tkinter. Suporta operações matemáticas básicas, funções trigonométricas, logaritmos e raízes quadradas, com interface gráfica amigável.

## Funcionalidades

- Operações básicas: +, -, *, /
- Potência: ^
- Raiz quadrada: sqrt()
- Logaritmo base 10: log()
- Logaritmo natural: ln()
- Funções trigonométricas: sin(), cos(), tan()
- Histórico da última operação
- Limpar visor (C) e apagar último caractere (⌫)

## Requisitos

- Python 3.6 ou superior
- Biblioteca padrão: tkinter, math (não é necessário instalar pacotes extras)
